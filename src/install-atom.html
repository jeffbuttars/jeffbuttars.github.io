Install the Atom text editor (Windows walkthrough)

Linux/Ubuntu download from here

Atom is a straight forward install

Head over to the Atom Download page here and download the installer

After the download finishes, open the installer by clicking on the download indicator on your web browser or by going to your Downloads folder and finding it there. You will then get a nice looking splash screen from the Atom installer while it finishes the installation.
After the installer has finished, open up the Atom editor. Typing Atom into the search/launcher input or find it in your applications directory. You should be greeted with the Atom editor looking something like the image below.
